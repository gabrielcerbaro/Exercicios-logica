## 📖 Sobre o Projeto

Este repositório reúne os exercícios de lógica que desenvolvo durante meus estudos de JavaScript.

O objetivo não é criar aplicações completas, mas fortalecer o raciocínio lógico e a capacidade de resolver problemas através do código. Cada desafio busca desenvolver a forma de pensar antes de programar, trabalhando conceitos fundamentais que serão utilizados em projetos mais complexos.

Os exercícios são realizados individualmente. A inteligência artificial é utilizada apenas para propor desafios, esclarecer dúvidas e revisar soluções quando necessário. Toda a lógica e a implementação das respostas são desenvolvidas por mim como parte do processo de aprendizado.
